SVMTutorial
===========

I. Introduction

This is a pdf version of "支持向量机通俗导论（理解SVM的三层境界）" generated with latex.

This tutorial is originally writtern by July and Pluskid in July's blog, namely, "结构之法算法之道blog" in CSDN.

II. Latex template
An off-the-shelf latex template from http://www.latextemplates.com/
template address is http://www.latextemplates.com/template/the-legrand-orange-book

To generate the pdf, you need to follow the instructions given by the template
1) pdflatex main
2) makeindex main.idx -s StyleInd.ist
3) biber main
4) pdflatex main x 2
